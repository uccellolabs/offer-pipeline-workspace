Conditions de viabilité

3 actions non négociables

Sans ces 3 conditions, l'offre reste viable sur le papier mais difficile à exécuter.

1
Discoverabilité à construire en priorité
Les PME en crise cherchent "comment gérer un prestataire qui ne livre pas" — pas "expert côté client digital". JF VILM doit être visible là où elles cherchent :
  • SEO long terme : articles "que faire quand votre prestataire web ne livre pas"
  • Prescripteurs EC (court terme) : contacter 3–5 cabinets avec proposition de partenariat
  • LinkedIn outreach (immédiat) : filtres gérants PME non-tech + détection signaux crise
2
3 premiers clients = cas fondateurs critiques
Sans social proof, le cycle de confiance est plus long. Sans ces 3 cas, chaque vente est 3× plus difficile.
  • Proposer un tarif découverte légèrement réduit (audit à 900 €) aux 3 premiers clients
  • En échange : témoignage écrit exploitable + autorisation de citer le cas (anonymisé)
  • Ces témoignages deviennent le levier de conversion de tous les suivants
  • Activer le réseau e-novateur en priorité pour trouver ces 3 premiers cas
3
Contrat type à rédiger AVANT le premier client
Risque scope creep documenté dans le profil Ikigai. Un contrat flou = première cliente = première déception.
  • Périmètre borné : ce qui est inclus / ce qui ne l'est pas
  • Livrables définis : rapport écrit, restitution 1h, plan d'action
  • Heures incluses + frais de dépassement explicites
  • Propriété des livrables : appartient au client dès paiement complet
  • Faire relire par un avocat PME avant utilisation

Plan d'action

Prochaines étapes recommandées

Court terme — 0-30 jours
  1. Rédiger le contrat type (périmètre, livrables, cap heures, frais dépassement) — faire relire par un avocat PME
  2. Créer le Calendly pour l'appel gratuit 30 min
  3. Activer le réseau e-novateur pour les 3 premiers clients fondateurs (tarif découverte 900 €)
  4. Déposer la candidature Activateur France Num (gratuit, label État)
Moyen terme — 30-90 jours
  1. Contacter 3–5 cabinets d'expertise comptable (proposition prescripteur 10%)
  2. Lancer le profil LinkedIn avec 3 posts thématiques "PME en crise prestataire digital"
  3. Valider le WTP réel avec les 3 premiers appels (ajuster pricing si résistance systématique)
Long terme — 90-180 jours
  1. Construire le SEO sur les mots-clés intention forte
  2. Revoir le pricing à la hausse dès 3 témoignages clients documentés
  3. Mission Stabilisation → 2 500–3 000 €/mois · Audit → 1 500–1 800 €
