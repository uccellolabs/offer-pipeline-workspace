# COLD_CALL_SCRIPT.md
- Date : 2026-05-03 | Version : 1.0 | Agent : offer-cold-call
- Cible : Consultant transformation digitale indépendant (Marc — ex-DSI, connaît Make, accompagne PME 20–80 salariés)
- Contexte : Froid — aucun contact préalable
- DISC estimé : Les 4 variantes (répartition CSV : C dominant ~44% | D ~30% | S ~16% | I ~10%)
- Volume : 5–20 calls/semaine

---

## Pitch en 1 phrase (brief interne — à intérioriser, pas à lire)

> "Hub IA souverain déployé en 1 journée chez leurs clients PME — 168€ de marge nette par client par mois, le support N2 reste chez nous."

---

## Script principal — Profil C (Conformité) ← le plus fréquent dans votre liste

*Signaux : profil LinkedIn détaillé avec certifications, posts techniques ou méthodologiques, questions précises, méfiance des superlatifs, ancienneté dans le même rôle.*

### Bloc 1 — Crochet (5 sec)

```
"Bonjour [Prénom], c'est Jonathan de Uccello Labs —
j'espère que je ne vous interromps pas au mauvais moment ?"
```

### Bloc 2 — Raison + légitimité (15 sec)

```
"Je vais être précis. Je travaille avec des consultants en
transformation digitale indépendants — profil Make ou intégration SaaS,
qui accompagnent des PME sur leurs outils.

On constate que la plupart n'ont pas de réponse structurée quand leurs
clients PME demandent de l'IA en garantissant que les données restent
chez eux."
```

### Bloc 3 — Question qualifiante (15 sec)

```
"Est-ce que la souveraineté des données IA chez vos clients —
et ce que vous leur proposez concrètement — c'est un angle
sur lequel vous avez une réponse aujourd'hui ?"
```

**Si oui →** Bloc 4
**Si non →** `"Qu'est-ce qui se passe de votre côté sur ce sujet ?"`
**Si "ça ne me concerne pas" →** `"D'accord. La question du revenu récurrent en complément de vos missions TJM, c'est un sujet sur lequel vous réfléchissez ?"`

### Bloc 4 — Proposition de valeur condensée (10 sec — si intérêt signalé)

```
"On est éditeur d'une plateforme hub IA — les consultants la revendent
à leurs clients PME. L'IA tourne sur l'infrastructure du client,
modèles open source, aucune donnée ne sort.

Les consultants qu'on accompagne constatent en moyenne 3 clients
en 60 jours sur un cycle de 2 à 4 semaines, avec une marge nette
documentée de 46 à 48%.

Je ne sais pas si ces chiffres correspondent à ce que vous cherchez —
c'est pour ça que je voulais en parler directement."
```

### Bloc 5 — Closing pour RDV (20 sec)

```
"Est-ce qu'on pourrait planifier 20 minutes pour que je vous montre
la méthodologie et qu'on vérifie si elle s'applique à votre profil
de clients — mardi ou jeudi ?"
```

**Si hésitation :**
```
"Pas d'engagement — on vérifie juste si c'est pertinent pour vous.
Si en 20 minutes ce n'est pas le cas, je ne vous recontacte pas."
```

|------|-----|-------------|---------------------|---------------------|
| **D** | Direct, chiffres, rapide | "Je serai bref." | "Sortir du tout-TJM, vous y mettez la pression cette année ?" | ROI chiffré + délai |
| **I** | Chaleureux, vision, storytelling | Référence à un post/sujet récent | "Productiser votre expertise — ça vous parle ?" | Transformation + liberté |

### Script D — Dominance

*Signaux : bio orientée résultats, KPI/ROI dans le vocabulaire, peu de posts personnels, ton direct, décisions rapides.*

**Règle d'or : aller vite, chiffres, pas d'émotion. Chaque phrase doit compter.**

```
"[Prénom], c'est Jonathan de Uccello Labs — je vous dérange ?"

[Réponse]

"Je serai bref. Je travaille avec des consultants transformation digitale
indépendants qui n'ont pas de MRR stable entre deux missions.

On leur donne un produit à revendre — 168€ de marge nette par client
par mois. Pas de support technique de leur côté.

Sortir du tout-TJM — c'est un sujet sur lequel vous avez mis la pression
cette année ?"
```

**[Oui] →**
```
"On est éditeur d'un hub IA souverain — hébergé chez le client, données
qui ne sortent pas. D'autres consultants ont signé 3 premiers clients
en 60 jours.

Vous avez 20 minutes jeudi ou vendredi pour voir si ça colle avec votre
profil de clients ?"
```

**[Non / Pas intéressé] →**
```
"D'accord. Le MRR c'est résolu chez vous, ou ça n'entre pas dans
votre modèle ?"
```

**Objection D — "Envoyez un email"**
```
"Je vais le faire. Avant : sortir du TJM, c'est votre priorité
ce trimestre ou plutôt H2 ?"
```

**Objection D — "Pas le temps"**
```
"Je comprends. 20 minutes — mardi ou jeudi, lequel vous convient ?"
```

### Script I — Influence

*Signaux : beaucoup de posts, emojis, partages fréquents, commentaires chaleureux, photo souriante, réseau large, contenu inspirationnel.*

**Règle d'or : chaleur, vision commune, jamais de sécheresse. Personnaliser si un post récent est visible.**

```
"Bonjour [Prénom] ! C'est Jonathan de Uccello Labs — je ne vous
dérange pas trop ?"

[Réponse]

[Si post récent visible] "J'ai vu votre post sur [sujet récent] —
vraiment juste ce que vous disiez."

"Je travaille avec des consultants en transformation digitale
comme vous — des gens qui ont vraiment la confiance de leurs clients
mais qui galèrent à transformer ça en quelque chose de récurrent.

On les aide à proposer une vraie offre IA souveraine à leurs clients
PME — quelque chose qui tourne sans eux et génère du MRR.

Est-ce que la question de productiser votre expertise — avoir quelque
chose qui rentre même quand vous êtes en vacances — c'est un sujet
qui vous parle en ce moment ?"
```

**[Oui] →**
```
"On est éditeur d'un hub IA — les données restent chez le client,
aucune fuite, le DSI signe. Vous revendez, on gère le support.

Ça change vraiment la dynamique pour les consultants avec qui on
travaille — certains ont ajouté 2 000€ de MRR en quelques mois
sans changer leur façon de bosser.

Je propose qu'on prenne 20 minutes pour vous montrer concrètement —
vous seriez plutôt dispo mardi ou jeudi ?"
```

**Objection I — "Pas le bon moment"**
```
"Je comprends les agendas. Juste pour voir si ça vaut de se
recontacter : la question du MRR, c'est quelque chose sur lequel
vous voulez avancer ce semestre ?"
```

**Objection I — "Envoyez un email"**
```
"Bien sûr ! Pour personnaliser : vous avez déjà essayé de productiser
quelque chose — diagnostic, formation — ou vous n'avez pas encore
sauté le pas ?"
```

### Script S — Stabilité

*Signaux : profil LinkedIn discret, peu de posts publics, rôle stable depuis longtemps, réponses courtes, prudent face aux nouvelles propositions.*

**Règle d'or : pas de pression, étapes claires, accompagnement visible. Ne pas brusquer.**

```
"Bonjour [Prénom], c'est Jonathan de Uccello Labs —
je ne vous dérange pas trop ?"

[Réponse]

"Merci. Je vais être bref — je travaille avec des consultants en
transformation digitale indépendants, et souvent on entend la même
chose : les missions se renouvellent, les clients font confiance,
mais si une mission se décale ou s'arrête, le mois est tendu.

Ce n'est pas simple à résoudre parce que ça demande d'avoir un produit
à proposer — pas juste du conseil.

Est-ce que c'est quelque chose que vous ressentez de votre côté ?"
```

**[Oui] →**
```
"On aide des consultants à revendre un hub IA souverain à leurs
clients PME — un produit clé en main, le support technique reste
chez nous. Ça crée du MRR sans changer leur façon de travailler.

Je ne sais pas si c'est pertinent pour vous — c'est pour ça
que je voulais vous poser la question.

Est-ce que vous auriez 20 minutes la semaine prochaine pour qu'on
échange tranquillement ? Pas d'offre commerciale — juste voir si
on peut vous être utile. Mardi matin ou jeudi après-midi ?"
```

**Objection S — "Je dois en parler à quelqu'un d'abord"**
```
"Bien sûr. Pour préparer cet échange : qu'est-ce qui vous a amené
à réfléchir à la question du MRR récemment ?"
```

**Objection S — "Pas le bon moment"**
```
"Aucun problème. C'est quand que ça aurait du sens pour vous —
ce trimestre ou plutôt après l'été ?
Je peux vous rappeler à ce moment-là si vous préférez."
```

### Script C — Conformité *(= script principal, voir ci-dessus)*

Rappel objections spécifiques C :

**Objection C — "Envoyez un email avec des références"**
```
"Tout à fait. Je vous envoie 2 cas avec les métriques précises.
Pour que ce soit pertinent : vous avez combien de clients PME actifs
en ce moment sur lesquels vous intervenez régulièrement ?"
```

**Objection C — "On fait déjà avec Make / on a déjà quelqu'un"**
```
"Je comprends. Notre différence par rapport à Make : l'IA tourne
sur l'infrastructure du client, vos clients posent leurs questions
en chat — ils n'ont rien à configurer, pas besoin d'un technicien
pour maintenir les workflows.

Est-ce que la souveraineté des données c'est un critère important
pour vos clients en ce moment ?"
```

**Objection C — "Vous avez des références ?"**
```
"Oui — je vous envoie les cas avec les chiffres. Et si vous voulez,
je peux vous donner accès à un sandbox pour tester avant qu'on
se reparle. Mardi ou jeudi, lequel vous convient ?"
```

## Script boîte vocale (20 sec max)

```
"Bonjour [Prénom], c'est Jonathan de Uccello Labs — [numéro].

Je travaille avec des consultants digitaux indépendants qui cherchent
à ajouter du MRR à leurs missions.

Je vous rappelle [jour et créneau précis] — ou vous pouvez me
rappeler au [numéro]."
```

*Répéter le numéro. Ton calme, confiant. Ne pas laisser plus de 2 messages.*

## Métriques de suivi — Volume 5–20 calls/semaine

| Métrique | Définition | Cible (votre niveau) |
|----------|-----------|---------------------|
| Taux de décroché | Appels décrochés / tentatives | 35–45% |
| Taux d'engagement | Prospects qui restent > 30 sec | 55–65% |
| Taux de conversion RDV | RDV pris / décrochés | 8–15% |
| Durée moyenne appel | Si décroché | 2–3 min |
| Calls pour 1 RDV | Tentatives nécessaires | 15–25 |
| Boîtes vocales | Max par prospect | 2 |

> Repère : sur 10 calls décrochés → 1 à 1,5 RDV avec un script calibré DISC.
> Dès la semaine 2, noter quelle objection revient le plus — ajuster le Bloc 3 en priorité.
