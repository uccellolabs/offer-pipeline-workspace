Modes de déploiement

Choisissez votre niveau de souveraineté.

Recommandé

SaaS souverain

L'IA tourne sur Scaleway (cloud souverain français). Votre base de données est isolée. Inclus dans tous les tiers.

Option

Scaleway (votre compte)

Vous maîtrisez votre propre compte Scaleway. Coût IA transparent et maîtrisable (~7–27€/mois selon tier). Idéal pour les PME RGPD sensibles.

Sur devis

On-premise

L'IA tourne intégralement sur votre infrastructure. Aucune donnée ne sort. Pour les secteurs réglementés, les données très sensibles ou les organisations sans tolérance cloud. Tarif sur devis selon votre configuration.

Questions fréquentes

Tout ce que vous voulez savoir.

  • Y a-t-il un engagement de durée ?
    Non. Uccello Hub fonctionne en abonnement mensuel sans engagement minimum. Vous pouvez arrêter à tout moment.
  • Les connecteurs supplémentaires sont-ils payants ?
    Au-delà du pack inclus dans votre tier, chaque connecteur supplémentaire est facturé 50€/mois. Le développement d'un nouveau connecteur non disponible prend ~24h.
  • Que se passe-t-il si je dépasse le nombre d'utilisateurs ?
    Vous pouvez passer au tier supérieur à tout moment. Il n'y a pas de facturation au dépassement — votre consultant vous prévient si vous approchez de la limite.
  • Le setup est-il compris dans l'abonnement ?
    Le setup (connexion des outils, paramétrage des droits, formation initiale) est facturé par votre consultant une seule fois à la signature. Comptez 500–1 500€ selon le nombre de connecteurs.
  • Que se passe-t-il si Uccello Labs disparaît ?
    Le contrat revendeur intègre des clauses de continuité de service. Si vous choisissez le mode on-premise, votre infrastructure vous appartient — vous ne dépendez pas de l'éditeur pour fonctionner.
  • Puis-je tester avant de m'engager ?
    Oui. Votre consultant peut organiser une démonstration sur votre propre stack avant toute signature. Le premier mois est offert.
