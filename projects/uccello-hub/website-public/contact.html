Avant la démo

Questions fréquentes.

  • Que dois-je préparer pour la démo ?
    Rien. Indiquez simplement quels outils vous utilisez (CRM, gestion de projet, messagerie…) lors de la prise de contact. Votre consultant se prépare de son côté.
  • Est-ce que mes données sont utilisées pendant la démo ?
    La démo peut se faire sur un environnement sandbox (données fictives) ou sur votre stack réel — c'est vous qui choisissez. Si vous optez pour le stack réel, votre consultant vous guide sur les accès nécessaires.
  • La démo est-elle payante ?
    Non. La démonstration est entièrement gratuite. Il n'y a aucun engagement avant la signature du contrat d'abonnement.
  • Dans quel délai puis-je obtenir une démo ?
    Votre consultant vous contacte sous 24h ouvrées pour planifier la démo. Selon les disponibilités, la démo peut être organisée dans les 48–72h suivant votre demande.
