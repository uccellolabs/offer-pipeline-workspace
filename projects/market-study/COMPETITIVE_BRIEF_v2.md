#### Fiche concurrent direct MS1 — Consultants n8n / Make

**Identité collective**
- Profil type : freelance senior automation, TJM 600-900€/jour
- Setup typique : install n8n self-hosted ou cloud + workflows custom + formation
- Ticket moyen par mission : 2-5k€

**Forces réelles**
- Marché établi, clients comprennent le concept "freelance n8n"
- Bonne reconnaissance LinkedIn et bouche-à-oreille fort dans la communauté NoCode FR

**Faiblesses documentées**
- Horizontal : font tout (CRM, marketing, finance), pas spécialisé "création d'offres"
- Workflows cousus main → fragiles, peu documentés, dépendent du consultant
- Pas de méthodologie business derrière → workflow = automation, pas un asset stratégique
- Marc connaît n8n, il peut bricoler — pas un service qui le sort de sa zone

**Adresse de Marc**
- ⚠️ Marc considère ces freelances mais leur reproche d'être "généralistes IA"
- ❌ Aucun ne propose un pipeline propriétaire éprouvé en production sur "création d'offres B2B"

#### Fiche concurrent direct MS3 — Frameworks open-source

**Identité collective**
- CrewAI, LangChain Templates, AutoGen, AgentGPT
- Templates gratuits, communauté active, GitHub stars

**Forces réelles**
- Gratuit
- Hautement personnalisable
- Communauté active

**Faiblesses documentées**
- Code à intégrer soi-même → 80h+ de setup pour un consultant non-dev
- Pas de méthodologie business (juste de la techno)
- Maintenance permanente (modèles, libs, breaking changes)
- 70% des projets perso "AI agents" abandonnés à 3 mois (pattern Reddit r/LocalLLaMA)

**Adresse de Marc**
- ✅ Tentation forte ("c'est gratuit + open + souverain")
- ❌ Mais Marc a déjà essayé un setup maison 18 mois → connaît le coût caché

#### Fiche concurrent direct HS1 — Big4 IA (Capgemini Invent / Onepoint AI)

**Identité**
- Cabinets IA grands groupes, équipes 100-1000+ AI consultants
- Positionnement : "transformation IA cabinet"
- Format : projet sur mesure, méthodologie propriétaire, multi-streams (techno + change + ops)

**Forces réelles**

**Adresse Hélène**

## Synthèse stratégique v2

**Aucun concurrent ne combine les 3 propositions de valeur de la gamme Offer Pipeline :**

1. **Pipeline propriétaire spécialisé "création d'offres B2B"** (vs concurrents généralistes IA)
2. **Modèle no-SaaS / souverain** (vs ChatGPT Enterprise / Copilot)
3. **3 portes d'entrée par profil** (livrable / install / studio)

**Risques concurrentiels à surveiller (12-24 mois) :**

- **Big4 IA** lance un "AI for consulting offer production" packagé → menace sur Pipeline Studio
  → mitigation : occuper le terrain dès Mois 3-6 avec 1-2 cabinets référents
- **Cabinet IA boutique** (Fabernovel/Datawise) packagise un produit "AI offer pipeline"
  → mitigation : référencer 1-2 clients pilotes vite
- **CrewAI / LangChain** publient un template "B2B offer pipeline" gratuit
  → mitigation : la valeur n'est pas le code, c'est la méthodologie + les prompts éprouvés —
  documenter la différence
- **Anthropic / OpenAI** publient un "Agent Builder for B2B" packagé
  → menace structurelle → mitigation : l'expertise installation + intégration CRM custom +
  formation reste hors de leur scope

**Décisions claires pour /pricing v2 :**

- Pipeline Solo entre formations IA (500-2.5k€) et consultants n8n (1.5-5k€) → cible 1990-2990€
- Pipeline Studio sous Big4 (60k+) et au-dessus intégrateurs Hubspot purs (5-30k€) →
  cible 6990-9990€ setup + custom + maintenance

**Décisions claires pour /offer-final v2 :**

- 3 phrases de positionnement distinctes (l'une ne remplace pas l'autre)
