## Type d’offre détecté

**B — Service** : forfaits mensuels + **audit** d’entrée + option **inbox** renforcée.

---

## Valeur économique (raisonnement)

- **1 mandat** (vente) commission typique = largement > coût annuel du service si un **seul** mandat provient d’un échange Instagram mieux traité.  
- *Aucun engagement sur résultat* — l’ancrage se fait sur **temps gagné** (dirigeant) + **risque d’invisibilité**.

**Prix value-based cible (France, indépendant crédible)** : fourchette **600–1 200€/mois** sur gestion + alignée presse pro CM ([Agence Netcom](https://agencenetcom.fr/community-manager-tarifs-reseaux/)) et packs spécialisés Insta ([exemple de grille](https://freelance-community-manager.com/tarifs-instagram/)).

## Architecture tarifaire (3 niveaux)

| | **Imm’Essentiel** | **Imm’Croissance** ⭐ | **Imm’Mandat** |
|---|:---:|:---:|:---:|
| **Prix (à partir de)** | **490€ / mois HT** | **790€ / mois HT** | **1 190€ / mois HT** |
| **Publications** | 8 posts/Stories planifiés | 12 + **2 Reels** simples / mois | 16 + **4 Reels** (dont **2 tournage smartphone** côté agence) |
| **Inbox** | 3j/sem 9h-18h commentaires + DM (SLA 24h ouvré) | 5j/sem sauf WE | 6j/sem + **priorité** DM < 4h ouvré |
| **Audit initial** | Inclus (light) | Inclus (complet) | + **atelier 90 min** équipe / mois |
| **Conformité légendes** | Checklist de base (checklist) | + relecture **systématique** annonces biens | + sync avec **fiche crédible** (template fourni) |

**Engagement minimum** : 3 mois (à afficher) — évite le churn « test 1 mois » sans effort.

**Audit seul (produit d’appel)** : **299€ HT (light) — 499€ HT (complet + plan 30j + call 60 min)**.
