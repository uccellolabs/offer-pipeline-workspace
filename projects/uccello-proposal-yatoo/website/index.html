Ce qu'on a compris du projet

On a étudié le cahier des charges YATOO. Voici notre lecture du besoin réel et comment on propose de l'attaquer.

Le problème

Un panneau LED qui n'est jamais à jour

Le CMS livré avec les panneaux est pensé pour des techniciens. Résultat : le restaurateur passe 15 minutes pour changer un texte, ou il ne le change plus du tout. L'investissement matériel est gâché.

✕ Interface desktop complexe, incompatible mobile
✕ Pas de génération IA : visuels faits à la main
✕ Gestion panneau par panneau, pas de multi-écrans
✕ Support en anglais, non adapté aux TPE françaises

La solution YATOO

90 secondes, smartphone, zéro formation

Une plateforme SaaS web-first, mobile-first, qui permet à n'importe quel professionnel de mettre à jour ses panneaux LED en 3 étapes depuis son téléphone, sans technicien.

✓ Interface Typeform (une étape à la fois)
✓ Génération IA de visuels brandés (Nano Banana)
✓ Multi-panneaux, diffusion WebSocket temps réel
✓ Setup QR code : opérationnel en 5 minutes
📱
App Flutter incluse dans le MVP. L'application Android fait partie du périmètre livré. Distribution via APK, installation manuelle sur les appareils des pilotes. Intégrée au backend Laravel en semaine 2, sans surcoût.

MVP · 2 semaines

Ce qu'on livre

Six modules couvrant le parcours complet, de la création de contenu à la diffusion sur le panneau, web et mobile.

🔐

Module 1

Auth multi-tenant

Compte par établissement. Login sécurisé, invitation collaborateur, gestion des droits.

Laravel 13 Sanctum
🎨

Module 2

Création de contenu

Flux Typeform (image, carrousel, vidéo). Upload S3. Templates métiers pré-configurés (restaurant, pharmacie, immo).

React Inertia S3
✨

Module 3

Génération IA

Intégration Nano Banana. Option A : 2 templates fixes. Option B : formulaire adaptatif avec champs automatiques selon le template. Visuels brandés générés en < 30s.

Nano Banana API
📡

Module 4

Connexion panneau

Setup QR code → auto-configuration. Diffusion WebSocket temps réel. Multi-panneaux (jusqu'à 5 en Pro). Fallback offline.

Reverb / Pusher WebSocket
📊

Module 5

Dashboard & SaaS

Dashboard web responsive (mobile-first). Historique des diffusions. Base multi-tenant pour abonnements futurs.

React Inertia PostgreSQL
📱

Module 6

App mobile Flutter

Application Android incluse dans le MVP. Installation via APK, sans passer par le Play Store. Connectée au backend Laravel via API. Diffusion sur panneau depuis le téléphone en 90 secondes.

Flutter Dart REST API
🚀
